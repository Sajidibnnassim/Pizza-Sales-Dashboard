# Pizza-Sales-Dashboard

🍕 Pizza Sales Analysis Dashboard
📌 Project Overview
This repository features a professional Power BI project designed to analyze retail pizza sales. The project transforms raw transactional data into business intelligence, helping stakeholders understand sales trends, customer preferences, and menu performance.

🚀 Key Performance Indicators (KPIs)
To measure business health, the following metrics were calculated using DAX:

Total Revenue: $817.86K – The sum of the total price of all pizza orders.

Average Order Value: $38.31 – The average amount spent per order.

Total Pizzas Sold: 49,574 – The sum of the quantities of all pizzas sold.

Total Orders: 21,350 – The total number of unique orders placed.

Avg Pizzas Per Order: 2.32 – The average number of pizzas sold per transaction.
