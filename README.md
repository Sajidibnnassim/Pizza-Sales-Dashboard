# Pizza-Sales-Dashboard

# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
This repository features a professional **Power BI** project designed to analyze retail pizza sales. The project transforms raw transactional data into business intelligence, helping stakeholders understand sales trends, customer preferences, and menu performance.

---

## 🚀 Key Performance Indicators (KPIs)
To measure business health, the following metrics were calculated using **DAX**:

* **Total Revenue:** `$817.86K` – The sum of the total price of all pizza orders.
* **Average Order Value:** `$38.31` – The average amount spent per order.
* **Total Pizzas Sold:** `49,574` – The sum of the quantities of all pizzas sold.
* **Total Orders:** `21,350` – The total number of unique orders placed.
* **Avg Pizzas Per Order:** `2.32` – The average number of pizzas sold per transaction.

---

## 🛠️ Tech Stack & Skills
* **Power BI Desktop:** Dashboard design, UI/UX, and interactive reporting.
* **Power Query:** Data cleaning, ETL (Extract, Transform, Load), and data type standardization.
* **DAX (Data Analysis Expressions):** Created advanced calculated measures and dynamic visuals.
* **Data Modeling:** Established relationships between tables to enable seamless cross-filtering.
* **SQL:** Validated data accuracy and performed initial exploratory data analysis (EDA).

---

## 📊 Dashboard Highlights & Insights

### 1. Sales Trends
* **Daily Trend:** Orders reach their peak on **Friday and Saturday evenings**, suggesting a need for increased staffing during weekends.
* **Monthly Trend:** Maximum order volume is observed in the months of **July and January**.

### 2. Product Performance
* **Category Leaders:** The **Classic Category** contributes the most to both total sales and total orders.
* **Size Preferences:** **Large size** pizzas are the most preferred by customers, driving the highest revenue share.
* **Inventory Optimization:**
    * 🏆 **Top Sellers:** The *Thai Chicken Pizza* and *Classic Deluxe Pizza* are top revenue generators.
    * ⚠️ **Bottom Sellers:** The *Brie Carre Pizza* and *Calabrese Pizza* are underperforming, indicating a need for menu review.
